# Netflix Content Analysis
## Overview

This project analyzes the Netflix Titles dataset using Python, Pandas and Matplotlib.
The dataset was cleaned and explored to identify trends in content types, ratings, genres, countries and release years.

### Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

### Key Findings
- Movies significantly outnumber TV Shows.
- Content releases increased rapidly after 2010.
- The United States contributes the most Netflix titles.
- TV-MA is the most common rating category.
- Dramas and International Movies are among the most common genres.
